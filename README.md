# IT-1430-A-M3
This is Assignment 3 for IT 2430 at Georgia Southern University. It builds on the HTML structure from Assignment 2 by adding an external CSS style sheet called amuse_styles.css to give the site a consistent and polished look. The site is for a fictional amusement park called Limitless Land. It includes four pages — a home page, a rides page, a park info page with a table, and a ticket order form. The styling follows a Japanese culture theme using colors inspired by torii gates, cherry blossoms, and traditional ink work. The main focus of this assignment was learning how to use external style sheets, HTML5 structural elements, and CSS properties like float, nth-child selectors, and form styling to create a cohesive and well-organized site.
